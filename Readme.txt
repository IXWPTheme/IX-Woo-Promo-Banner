IX Woo Pro Banner is a Woocommerce Promo Banner Optimized Plugin for DomPDF Generator.

== Description ==

A powerful WooCommerce extension that creates professional promotional banners with PDF generation capabilities.

== Features ==
- Custom promotional banners for products
- PDF generation with DomPDF
- Flexible shortcode implementation
- Responsive design
- Admin management interface

== Installation ==
1. Upload the plugin files to the `wp-contentpluginsix-woo-pro-banner` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure settings under WooCommerce  IX Pro Banner
